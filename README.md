## Project Overview

This is a simple Image Gallery project built using HTML and CSS (Flexbox).
The gallery displays images in a clean grid layout with 4 images per row.
It is designed to practice Flexbox layout, alignment, spacing, and styling.

## Features

Responsive Flexbox layout

4 images per row

Clean and centered design

Image borders and spacing

Styled heading with underline

Simple and beginner-friendly code

## Technologies Used

HTML5

CSS3 (Flexbox)
